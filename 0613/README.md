## ディレクトリ構成

```
App/
 ├ backend/             // サーバーとかデータベースとか公開しないコード
 │  ├ routes/           // API
 ├ public/              // HTML,CSS,JSの公開するコード
 │  ├ models/           // Class
 │  └ pages/            // 各ページ
 └ README.md            // メモ
```
